# Rotwood Korean Translation

Open-source Korean translation project for Rotwood, maintained by contributors.

## About
This repository is a community-driven Korean translation project for Rotwood.
Its purpose is to organize, review, and improve Korean translations in an open and collaborative way.

## Goals
- Provide a complete Korean translation for Rotwood
- Allow contributors to suggest and review translations together
- Keep translation files updated as source strings change
- Maintain terminology consistency with shared glossary rules

## Contribution
Contributors can help by:
- translating new strings
- reviewing existing translations
- suggesting better wording
- reporting terminology inconsistencies
